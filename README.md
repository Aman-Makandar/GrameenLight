# Grameen-Light

Grameen-Light is an Android application developed using Kotlin and Jetpack Compose for smart rural infrastructure and utility management. The application integrates Firebase services and Google Maps to provide efficient monitoring and management of electrical poles and related infrastructure.

## Features

- Google Sign-In Authentication
- Google Maps Integration
- Firebase Firestore Database
- MVVM Clean Architecture
- Real-time Pole Monitoring
- Modern Jetpack Compose UI
- Secure Cloud Data Storage
- Responsive Android Design

## Tech Stack

- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore
- Google Maps API
- MVVM Architecture
- Android Studio

## Project Architecture

The application follows MVVM Clean Architecture to maintain scalability, maintainability, and separation of concerns.

### Layers
- UI Layer
- ViewModel Layer
- Repository Layer
- Firebase Backend Layer

## How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/Aman-Makandar/GrameenLight.git
```

2. Open the project in Android Studio

3. Add your Firebase `google-services.json` file

4. Sync Gradle files

5. Run the application on emulator or physical device
